Tabla usando Grid

Cuadrícula

Una cuadrícula es un conjunto de líneas horizontales y verticales que se intersectan, definiendo columnas y filas.

Referencia

Tamaños de trazado

Puedes crear una cuadrícula con tamaños fijos (píxeles) o flexibles (porcentajes o unidades fr).

Referencia

Posicionamiento de elementos

Los elementos se pueden colocar en ubicaciones precisas dentro de la cuadrícula usando números de línea, nombres o seleccionando un área.

Referencia

Alineación y superposición

Grid permite controlar la alineación de los elementos y la superposición de contenido usando la propiedad z-index.
