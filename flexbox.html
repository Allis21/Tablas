Tabla usando Flexbox

Flex Container y Flex Items

El contenedor flex se define con display: flex; y los elementos hijos dentro de este contenedor son flex items.

Referencia

Ejes Principal y Cruzado

El eje principal se define con flex-direction y puede ser horizontal (row) o vertical (column). El eje cruzado es perpendicular al eje principal.

Referencia

Justificar Contenido

La propiedad justify-content alinea los flex items a lo largo del eje principal. Las opciones incluyen flex-start, flex-end, center, space-between, y space-around.

Referencia

Alinear Items

La propiedad align-items alinea los flex items a lo largo del eje cruzado. Las opciones incluyen flex-start, flex-end, center, baseline, y stretch.
